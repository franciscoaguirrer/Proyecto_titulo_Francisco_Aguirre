# Making Trips – Sistema de Gestión de Transporte y Turismo

Sistema web desarrollado como proyecto de titulación, orientado a la gestión de servicios de transporte de pasajeros y turismo.

## Tecnologías utilizadas
- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express
- Base de datos: MongoDB Atlas
- Autenticación: JWT
- Control de roles: Administrador / Operador

## Estructura del proyecto
- /frontend → Aplicación web cliente
- /backend → API REST y lógica de negocio

## Instalación y ejecución

### Requisitos
- Node.js
- NPM

### Backend

- cd backend
- npm install
- npm run dev

### Frontend

- cd frontend
- npm install
- npm run dev
